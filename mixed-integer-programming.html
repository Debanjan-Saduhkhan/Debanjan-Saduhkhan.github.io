Mixed Integer Linear Programming

Optimization algorithms, Resource allocation, Constraint programming

Research Overview

My research in Mixed Integer Linear Programming focuses on developing efficient optimization algorithms for complex resource allocation problems in wireless sensor networks and smart building management. I specialize in constraint programming, branch-and-bound algorithms, and cutting plane methods for solving large-scale optimization problems with discrete and continuous variables.

Key Publications

Optimal HVAC Control for Smart Building Management

Research Report

This research focuses on optimizing HVAC control systems using mixed-integer linear programming for smart building management applications. The work demonstrates how MILP can be integrated with machine learning techniques for energy-efficient building operations and optimal resource allocation.

Key Contributions:

  • Mixed-integer linear programming for HVAC optimization
  • Smart building management applications
  • Integration of MILP with machine learning

Minimum Cost Event-Driven WSN with Spatial Differentiated QoS Requirements

Wireless Networks, Springer 2018

This paper presents a minimum cost approach for event-driven wireless sensor networks with spatial differentiated quality of service requirements using mixed integer programming techniques. The work optimizes network deployment and operation costs while meeting varying QoS demands across different spatial regions.

Key Contributions:

  • Mixed integer programming for WSN optimization
  • Spatial differentiated QoS constraints
  • Cost-effective network deployment
  • QoS-aware resource allocation
  • Constraint programming for network design

Energy Efficient Multi-Beacon Guard Method for Periodic Data Gathering in Time Synchronized WSN

Wireless Networks, Springer 2020

This paper presents an energy-efficient multi-beacon guard method for periodic data gathering in time synchronized wireless sensor networks using optimization techniques. The work focuses on optimizing energy consumption while maintaining reliable data collection through constraint-based programming.
