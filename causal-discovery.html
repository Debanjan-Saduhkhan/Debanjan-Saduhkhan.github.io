Causal Discovery

Causal inference, Structural equation modeling

Research Overview

My research in Causal Discovery focuses on developing novel techniques to discover causal relationships in volatile user behavior patterns. I specialize in structural equation modeling and causal inference methods that enable interpretable AI solutions for personalized treatments and engagement strategies.

Key Publications

EFfECT-RL: Enabling Framework for Establishing Causality and Triggering Engagement through RL

ACM CIKM 2024

This paper presents a novel causal discovery technique for volatile user gameplay behavior. The framework establishes causality in complex user interaction patterns and uses this understanding to generate personalized treatments through reinforcement learning.

Key Contributions:

  • Novel causal discovery technique for volatile user behavior
  • Framework for establishing causality in complex interactions
  • Personalized treatment generation through causal understanding
  • Integration of causal discovery with reinforcement learning
View on ACM Digital Library Download PDF

Generalized Experiment Conclusion Framework for Extremely Volatile Data

ACM IKDD CODS and COMAD 2024

This work addresses the challenge of drawing reliable conclusions from extremely volatile data through a generalized experiment conclusion framework. The approach provides robust methods for causal inference in highly dynamic environments.

Key Contributions:

  • Generalized framework for volatile data analysis
  • Robust causal inference in dynamic environments
  • Reliable experiment conclusion methods
  • Applications in highly volatile user behavior
